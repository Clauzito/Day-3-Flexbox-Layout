What I practiced

- HTML semantic structure (header, main, section, aside, footer)
- Flexbox layout for page structure
- Navigation bar styling
- Hover effects on links
- CSS transitions and transform (scale animation)
- Using `<figure>` and `<figcaption>` for images
- `<details>` element for interactive content

---

## 🎯 Goal of this project

Understand how to build a simple web page layout using Flexbox and improve basic CSS styling skills.
